# 🎙️ Audio Recorder

Професійний аудіо рекордер з красивим графічним інтерфейсом, написаний на C++ з використанням Dear ImGui та PortAudio.

## ✨ Особливості

- 🎤 **Запис аудіо** з мікрофона у форматі WAV
- 🎨 **Сучасний UI** з темною затишною темою
- 📊 **Візуалізація рівня звуку** в реальному часі
- 🔧 **Налаштування**: вибір пристрою вводу, частота дискретизації, кількість каналів
- ⏱️ **Таймер запису** з відображенням часу
- 📈 **Історія рівня звуку** у вигляді графіка

## 🛠️ Технології

- **C++17** - основна мова програмування
- **PortAudio** - кросплатформна бібліотека для роботи з аудіо
- **Dear ImGui** - бібліотека для створення GUI
- **GLFW** - бібліотека для роботи з вікнами та OpenGL
- **CMake** - система збірки

## 📋 Вимоги

- CMake 3.15 або новіше
- C++17 сумісний компілятор (MSVC, GCC, Clang)
- OpenGL 3.3+


## 📦 Використання

1. Запустіть програму `AudioRecorder`
2. Виберіть пристрій вводу зі списку
3. Введіть ім'я файлу для збереження (за замовчуванням `recording.wav`)
4. Натисніть "⏺️ Start Recording" для початку запису
5. Натисніть "⏹️ Stop Recording" для зупинки
6. Файл буде збережено у поточній директорії

## ⚙️ Налаштування

У меню "Settings" можна змінити:
- **Sample Rate**: 22050 Hz, 44100 Hz, 48000 Hz, 96000 Hz
- **Channels**: Mono або Stereo

*Примітка: налаштування можна змінювати тільки коли запис не активний*
